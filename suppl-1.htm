Description

SampleModel.txt is an example of BUGS code for one of the models in the paper (the final model for Douglas-fir as selected in the forward-selection procedure). BUGS.data.DF.txt is an ASCII file which contains the data for Douglas-fir, formatted for BUGS. BUGS.inits.DF.txt is an ASCII file which contains initial values for the latent states and parameters, formatted for BUGS. Because MCMC mixing for complex models can be sensitive to initial conditions, we recommend using these initial values rather than allowing BUGS to generate initial values.
